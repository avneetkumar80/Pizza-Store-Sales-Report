# 🍕 Pizza Sales Analysis – Power BI Dashboard

## 📌 Project Overview
The **Pizza Sales Analysis** project is an end-to-end data analytics solution built using **Power BI** to analyze sales performance of a pizza store.  
The dashboard converts raw transactional data into **interactive visual insights** that help understand customer behavior, sales trends, and product performance.

This project is designed to support **business decision-making** by identifying revenue drivers, demand patterns, and underperforming products.

---

## 🎯 Business Objectives
- Track overall sales and revenue performance  
- Identify top-performing and low-performing pizzas  
- Analyze customer ordering patterns  
- Study seasonal and monthly sales trends  
- Support data-driven pricing and inventory decisions  

---

## 📊 Key Performance Indicators (KPIs)
The following KPIs were calculated using **DAX measures**:

- **Total Revenue** – Total income generated from pizza sales  
- **Total Orders** – Count of unique customer orders  
- **Total Pizzas Sold** – Total quantity sold  
- **Average Order Value** – Revenue per order  
- **Average Pizzas per Order** – Customer buying behavior metric  

---

## 📈 Dashboard Analysis & Insights

### 🔹 Sales Trends
- Daily and monthly sales trends highlight peak and low-demand periods  
- Seasonal patterns help identify high-revenue months  

### 🔹 Product Performance
- Top 5 best-selling pizzas by quantity and revenue  
- Bottom 5 least-selling pizzas for optimization or removal  

### 🔹 Category & Size Analysis
- Comparison of sales across pizza categories (Classic, Veggie, etc.)  
- Performance analysis by pizza size (Small, Medium, Large)  

---

## 🧠 Data Modeling & DAX
- Relationships created between date, order, and sales tables  
- Custom DAX measures used for:
  - Revenue calculations
  - Order-level analysis
  - Average metrics
- Filters and slicers optimized for fast interactivity  

---

## 🛠 Tools & Technologies
- **Power BI Desktop** – Dashboard creation and visualization  
- **DAX** – Business logic and KPI calculations  
- **Excel / CSV** – Data source and preprocessing  
- **Git & GitHub** – Version control and project hosting  

---

## 🧩 Dashboard Features
- Interactive slicers (Date, Category, Size)  
- Dynamic KPI cards  
- Minimalistic and business-friendly design  
- User-friendly navigation  
- Real-time visual filtering
